Two-point functions

B0i(id, p^2, m1^2, m2^2)

two-point tensor coefficient for id

\[\frac{\mu^{4-D}}{i\pi^{D/2} r_\Gamma} \int \frac{({\rm numerator})\, d^D q }{(q^2-m_1^2)\left[(q+p)^2-m_2^2\right]} \quad{\rm with}\quad r_\Gamma = \frac{\Gamma^2(1-\varepsilon)\Gamma(1+\varepsilon)}{\Gamma(1-2\varepsilon)},~D=4-2\varepsilon.\]

Special cases:

B0(p^2, m1^2, m2^2)

the scalar two-point one-loop function

\[\frac{\mu^{4-D}}{i\pi^{D/2} r_\Gamma} \int \frac{d^D q }{(q^2-m_1^2)\left[(q+p)^2-m_2^2\right]} \quad{\rm with}\quad r_\Gamma = \frac{\Gamma^2(1-\varepsilon)\Gamma(1+\varepsilon)}{\Gamma(1-2\varepsilon)},~D=4-2\varepsilon.\]
