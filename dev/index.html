LoopTools.A0 — Method
A0(m^2)

the scalar one-point one-loop function $\dfrac{μ^{4-D}}{iπ^{D/2} r_Γ} \int d^D q \dfrac{1}{q^2-m^2}$ with $r_Γ = \dfrac{Γ^2(1-ε)Γ(1+ε)}{Γ(1-2ε)}$, $D=4-2ε$.

LoopTools.A0i — Function
A0i(id, m^2)

one-point tensor coefficient for id

\[\frac{μ^{4-D}}{iπ^{D/2} r_Γ} \int d^D q \frac{\{1, g_{μν} \} }{q^2-m^2} \quad{\rm with}\quad r_Γ = \frac{Γ^2(1-ε)Γ(1+ε)}{Γ(1-2ε)},~D=4-2ε.\]

LoopTools.B0 — Method
B0(p^2, m1^2, m2^2)

the scalar two-point one-loop function

\[\frac{μ^{4-D}}{iπ^{D/2} r_Γ} \int \frac{d^D q }{(q^2-m_1^2)\left[(q+p)^2-m_2^2\right]} \quad{\rm with}\quad r_Γ = \frac{Γ^2(1-ε)Γ(1+ε)}{Γ(1-2ε)},~D=4-2ε.\]
