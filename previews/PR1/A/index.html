One-point Functions

A0i(id, m^2)

one-point tensor coefficient for id

\[\frac{\mu^{4-D}}{i\pi^{D/2} r_\Gamma}\]
\[\frac{\mu^{4-D}}{i\pi^{D/2} r_\Gamma} \int d^D q \frac{\{1, g_{\mu\nu} \} }{q^2-m^2} \quad{\rm with}\quad r_\Gamma = \frac{\Gamma^2(1-\varepsilon)\Gamma(1+\varepsilon)}{\Gamma(1-2\varepsilon)},~D=4-2\varepsilon.\]

Special cases:

idIntDescription
aa01scalar one-point one-loop function, i.e., A0(m^2)
aa004coefficient of $g_{\mu\nu}$

A0(m^2)

the scalar one-point one-loop function $\\dfrac{\mu^{4-D}}{i\pi^{D/2} r_\Gamma} \\int d^D q \\dfrac{1}{q^2-m^2}$ with $r_\Gamma = \\dfrac{\Gamma^2(1-\varepsilon)\Gamma(1+\varepsilon)}{\Gamma(1-2\varepsilon)}$, $D=4-2\varepsilon$.
