Three-point functions

C0i(id, p1^2, p2^2, (p1+p2)^2, m1^2, m2^2, m3^2)

three-point tensor coefficient for id

\[\frac{\mu^{4-D}}{i\pi^{D/2} r_\Gamma} \int \frac{({\rm numerator})\, d^D q }{(q^2-m_1^2)\left[(q+p_1)^2-m_2^2\right] \left[(q+p_1+p_2)^2-m_3^2\right]} \quad{\rm with}\quad r_\Gamma = \frac{\Gamma^2(1-\varepsilon)\Gamma(1+\varepsilon)}{\Gamma(1-2\varepsilon)},~D=4-2\varepsilon.\]
